== Installation ==

Installing this plugin as regular wordpress plugin.

After install, this plugin will enable a custom post called <strong>Coupons</strong>. Add category & coupon items on that custom post. 
After adding items, you are ready to use this plugin in your post, page, widget or anywhere you like. 

Go to page > Add New. On the editor, you will see a button called <strong>Coupon Shortcode</strong> by clicking button you see the shortcode settings popup, enter category name on first field, not category id or slug. Make sure spelling is ok. It is case sensitive. 

Here is the default shortcode <code> [coupon category="coupun-category" border_color="#dddddd" title_color="#0099FF" title_font_size="21" desc_text_color="#868686" other_text_color="#a1c44e" btn_style="blue"] </code>
